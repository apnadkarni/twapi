Tcl Windows Help

This is a compilation of the documentation for Tcl/Tk and the most commonly used extensions and libraries on Windows. Advantages over various other documentation sources are:

  • Includes Tcl/Tk, standard libraries and the most commonly used Windows extensions in a single help file.
  • Comprehensive table of contents. Commands or C functions within a single page have individual entries. Subcommands, (for example, string match), widget options (for example, -activebackground), and important details (for example, %f focus specifier) have their own entries for ease of lookup.
  • A combined and detailed index makes it easy to locate specific information. For example, typing try in the index input box will bring up links to both the TWAPI and Tcl versions of the command. Similarly, typing %EE will allow quick access to the detail for how this specifier is handled by the clock command and typing <<MenuSelection>> will take you to the description for that event. Together with incremental lookup, this makes locating information very quick and easy.
  • Full text content search.

Documentation for the following distributions are included:

  • Tcl/Tk 8.6 including TDBC and Itcl
  • Tcl Windows API 3.0
  • Tcllib 1.12
  • Tklib 0.5
